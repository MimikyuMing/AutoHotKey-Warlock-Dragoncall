
# 使用须知
# 前提
- 布局默认(双排布局)
- 大小默认
- 亮度50
- 金v需要在config.ini设置true,非金书则是false
# 按键须知
1. [Ctrl + F11] 启动脚本
2. [Ctrl + F5] 重载脚本
3. [XButton2] 鼠标侧键2 持续触发
4. [Ctrl + F3] 显示/隐藏GUI
5. [XButton1] 鼠标侧键1 闪避(ss) 

# 更新日志3.3

1. 重构了状态管理模块,现在引入了Buffs的状态管理,从原本的固定前三个buff识别扩展到20个,并且可根据config.ini文件进行自定义
2. 图像识别方面,从原来的两次GetRegionColors()优化成一次检测,减少了50%的检测时间
3. 由于状态管理的重构,导致原本的SoulFlare/LeechSubscribers监听者无法使用,因此重构了逻辑,在SkillStatusCheck()方法中,加入了or运算,当检测到指定buff的时候直接break循环,并且notity监听者
4. 对大部分参数进行了可配置化,为日后可动态识别游戏界面以适配而打下基础
