# 基于AutoHotKey的 剑灵neo 的咒术暴魔灵流派宏

AHK 版本需要v2.0

# 使用须知
# 前提
- 布局默认(双排布局)
- 大小默认
- 亮度50
# 按键须知
1. [Ctrl + F11] 启动脚本
2. [Ctrl + F5] 重载脚本
3. [XButton2] 鼠标侧键2 持续触发
4. [Ctrl + F3] 显示/隐藏GUI
5. [XButton1] 鼠标侧键1 闪避(ss) 


# 3.3更新须知

**3.3可能不稳定,请谨慎使用**

1. 重构了状态管理模块,现在引入了Buffs的状态管理,从原本的固定前三个buff识别扩展到20个,并且可根据config.ini文件进行自定义
