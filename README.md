# Warlock_Dragoncall No48 251025.ahk
- ctrl + f11 启动/禁用宏
- XButton2 卡刀触发键（长按）（需Ctrl+f11启动）
- XButton1 一键ss（需Ctrl+f11启动）
- Ctrl + f5 重启宏（需Ctrl+f11启动）

内力低于6点自动使用真言，低于3点使用破裂
降临情况下，低于4点使用真言，低于1点使用破裂
